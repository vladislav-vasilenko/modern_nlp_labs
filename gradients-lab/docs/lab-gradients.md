## 2. Затухание и взрыв градиентов в глубоких сетях

В глубоких нейросетях при умножении матриц весов на каждом слое градиент может уменьшаться по экспоненте (затухать) или увеличиваться по экспоненте (взрываться). Это зависит от:
1. **Инициализации весов**: стандартное нормальное распределение $\mathcal{N}(0, 1)$ vs Xavier (Glorot) / Kaiming (He).
2. **Функций активации**: сжимающие функции (Sigmoid, Tanh) с малой производной vs ReLU.

## 3. Клиппинг градиентов (Gradient Clipping)

Если градиенты взрываются, это приводит к числовой нестабильности (`NaN` при обучении). Самый простой способ борьбы с этим — принудительное ограничение (клиппинг) градиентов перед шагом оптимизатора. Различают два метода:
1. **Clipping by value**: ограничение каждого элемента градиента в диапазон $[-c, c]$.
2. **Clipping by norm**: масштабирование всего вектора градиентов всех параметров так, чтобы его суммарная $L_2$-норма не превышала заданный порог $M$.

3. Сравните работу вашей ручной реализации клиппинга с методами PyTorch (`nn.utils.clip_grad_value_` и `nn.utils.clip_grad_norm_`).
